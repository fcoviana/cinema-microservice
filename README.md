# Cinema Microservices
Tem como objetivo exemplificar a criação de micro-serviços com NodeJS.

---

#### Arquitetura

<div align="center"><a href="#"><img width="722" height="265" src="./assets/img/consulta-de-lancamentos.png"></a></div>

---

#### Dependências
- NodeJs
- yarn || npm
- MongoDB
- express
- helmet
- morgan
- supertest

---

#### Comandos
###### Executar Testes
```shell
    npm test
```

###### Executar servidor
```shell
    npm start
```
